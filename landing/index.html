⚠️ macOS sagt „Dream OS ist beschädigt"? Ganz normal in der Beta — hier die Lösung.

Weil wir (noch) kein 99-€-Apple-Zertifikat haben, blockt macOS unbekannte Apps. Kein echter Schaden. So geht's:
1. Zip entpacken (Doppelklick) → daraus wird Dream OS.app · 2. App in den Ordner Programme oder Downloads legen · 3. Terminal öffnen (Cmd+Leertaste → „Terminal"), diese Zeile einfügen & Enter:

xattr -cr ~/Downloads/Dream\ OS.app

Wichtig: Es kommt keine Meldung zurück — genau das heißt „hat geklappt". Kein Fehler = Erfolg.
4. Dann Rechtsklick auf die App → Öffnen → nochmal Öffnen. Ab jetzt startet sie normal per Doppelklick.
Windows: „Windows hat den PC geschützt" → Weitere Informationen → Trotzdem ausführen.

Für die lokale KI einmal Ollama installieren (gratis) → ollama pull llama3.1:8b. Läuft danach komplett offline auf deinem Rechner.
